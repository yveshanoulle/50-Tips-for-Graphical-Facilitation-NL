# 50-Tips-for-Graphical-Facilitation-NL

Het project voor de nederlandse vertaling.

De bedoeling is om per pagina een vertaling op te laden. Om zeker te zijn dat andere zien dat je aan een pagina bezig bent, 
kan ik je aanraden als je aan een pagina begint, om onmiddelijk je naam in die pagina te zetten en op te laden. 
(tenzij iemand een beter idee heeft.)

pagina structuur:

page001_001_Title.txt origineel

page001_002_title.txt vertaling

page001_003_title.txt  eerste review

page001_004_title.txt tweede review

Om het voor reviewers makkelijker te maken, gaan we de Engelse tekst onder de Nederlandse laten staan.

Als je een pagina hebt vertaald voeg zeker ook je naam toe aan vertalers.txt

in a_glossary verzamelen we onze typische vertalingen.
